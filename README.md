# ninja

Reverse engineered `ChatGPT` proxy

> Project has ended.

### Features

- API key acquisition
- `Email`/`password` account authentication
- Proxy `ChatGPT-API`/`OpenAI-API`
- ChatGPT WebUI
- Support IP proxy pool
- Very small memory footprint
